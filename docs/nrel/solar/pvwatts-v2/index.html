PVWatts (Version 2 - Deprecated) (GET /api/georeserv/app/sam/pvwatts)

Deprecated: We encourage you to migrate to version 4 of the PVWatts API.

Returns the results for a PVWatts simulation, including financial and utility data.

    Request URL

    GET /api/georeserv/app/sam/pvwatts.format?parameters

    Request Parameters

    Parameter Required Value Description
    format Yes
    Type: string
    Default: None
    Options: json, xml, csv

    The output response format. Given in the URL as the file extension.
    api_key Yes
    Type: string
    Default: None

    Your developer API key. See API keys for more information.
    timeframe No
    Type: string
    Default: daily
    Options: monthly, daily

    Granularity of the output response
    lat Depends
    Type: decimal
    Default: None

    lon Depends
    Type: decimal
    Default: None

    address Depends

    Default: None

    climate_file_type No
