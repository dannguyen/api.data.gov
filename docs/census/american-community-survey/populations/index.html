Populations - All States

Return the populations of all states.

    Request URL

    GET http://api.data.gov/census/american-community-survey/v1/{year}/populations/states?parameters

    Request Parameters

    Parameter Required Value Description
    year Yes
    Type: integer
    Default: None
    Options: 2010, 2011

    The American Community Survey year to pick.
    api_key Yes
    Type: string
    Default: None

    Your developer API key. See API keys for more information.

    Examples

    JSON Output Format

    GET /census/american-community-survey/v1/2011/populations/states?api_key=DEMO_KEY
    [
  ["B02001_001E","NAME","state"],
  ["4747424","Alabama","01"],
  ["700703","Alaska","02"],
  ["6337373","Arizona","04"],
  ["2895928","Arkansas","05"],
  ["36969200","California","06"],
  ...
]

    Rate Limits

    Standard rate limits apply. No more than 1,000 requests may be made in any hour. No more than 10,000 requests may be made in any day.

    Errors

    Standard errors may be returned.
